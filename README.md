# hdririg
A project consisting of a hardware software package in order to create HDRIs using your mobile phone.


# Technical:
If you want to run the script inside blender. You need to create a viritual environment in the same folder as the .blend file. You can create the environment by: `python -m venv venv` followed by: `./venv/Script/activate/` (on windows) then install the dependecie(s) by `pip install ezdxf`
